# 🚀 InterviewIQ - AI Interview Preparation Platform

> A full-stack AI-powered mock interview platform that helps users practice technical and HR interviews with personalized AI-generated questions and feedback.

🌐 **Live Demo:** https://interviewiq-52sz.onrender.com/

---

## 📖 About the Project

InterviewIQ is a modern AI interview preparation platform where users can:

- 🔐 Create an account and securely log in
- 🤖 Practice AI-generated mock interviews
- 📄 Upload their resume
- 💬 Answer interview questions
- 📊 Receive AI-powered interview feedback
- 📈 Track interview performance

The project is built using the **MERN Stack** and integrates **Google Gemini AI** to simulate realistic interview experiences.

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/AnujGangwar01/Interview-Agent.git
```

Move inside the project

```bash
cd Interview-Agent
```

---

## Backend Setup

```bash
cd server

npm install
```

Create a `.env` file

```env
PORT=5000

MONGO_URI=Your MongoDB Atlas URI

JWT_SECRET=Your Secret Key

GEMINI_API_KEY=Your Gemini API Key
```

Run backend

```bash
npm run dev
```

---

## Frontend Setup

```bash
cd client

npm install

npm run dev
```
